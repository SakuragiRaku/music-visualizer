ミュージックビジュアライザー 仕様書


============================================================
概要
============================================================

アプリ名:   ミュージックビジュアライザー
ジャンル:   音楽可視化ツール

どんなアプリ?
  PCのスピーカーから出ている音をリアルタイムに幾何学模様として可視化するアプリ。
  Chrome の getDisplayMedia API でタブの音声を取得し、
  Web Audio API の AnalyserNode で周波数解析し、
  Canvas にビジュアルエフェクトを描画する。
  波形、パーティクル、万華鏡など複数のモードを搭載。

音声の取得方法
  ブラウザのセキュリティ制約上、直接スピーカー出力は取得できない。
  代わりに getDisplayMedia API で「タブの音声を共有」する。
  ユーザーは音楽を再生しているタブを選び、音声共有を有効にする。
  これにより、マイクではなくスピーカー出力の音のみに反応する。

誰が使う?
  音楽を聴きながら映像を楽しみたい人、配信の背景に使いたい人。


============================================================
機能一覧
============================================================

音声キャプチャ
  getDisplayMedia API でタブ音声を取得
  AnalyserNode で周波数データをリアルタイム解析
  音声取得の開始/停止ボタン

ビジュアルモード (5種類)
  バー:       縦棒が周波数帯ごとの音量に反応
  ウェーブ:   波形データを滑らかな曲線で描画
  サークル:   円形に配置された棒が中心から外に伸縮
  パーティクル: 音量に応じてパーティクルが飛び散る
  万華鏡:    対称的な幾何学模様がリアルタイムに変形

カラーテーマ (4種類)
  レインボー:  周波数帯に応じてHSLで自動着色
  ネオン:      シアン + マゼンタ + イエローのネオンカラー
  モノクロ:    白一色、シンプルで美しい
  オーシャン:  青〜ティール系のグラデーション

設定
  FFTサイズ変更 (解像度切替: 低/中/高)
  感度調整スライダー
  背景の透明度 (軌跡を残す効果)
  フルスクリーン切り替え

タブ映像背景 (オーバーレイ)
  共有中のタブの映像を半透明で背景に表示
  不透明度スライダーで透過度を調整可能
  ON/OFFトグルボタン付き
  ビジュアライザーが映像の上に重なって描画される

UI
  フローティングのコントロールパネル (自動隠し)
  マウス移動で表示、3秒操作なしで非表示
  キーボードショートカット (スペースで開始/停止、1-5でモード切替)


============================================================
画面構成
============================================================

メイン画面
  フルスクリーンCanvas (ビジュアル描画)
  中央に大きな開始ボタン (初期状態)
  フローティングコントロールパネル (下部)

コントロールパネル (フローティング)
  音声取得 開始/停止ボタン
  モード切替ボタン (5種)
  カラーテーマ切替ボタン (4種)
  感度スライダー
  フルスクリーンボタン
  設定ボタン

設定パネル (スライドイン)
  FFTサイズ選択
  感度スライダー
  背景透明度スライダー


============================================================
技術構成
============================================================

フレームワーク: なし (HTML / CSS / JavaScript)
データ永続化:  LocalStorage (設定のみ)
音声取得:      getDisplayMedia API
音声解析:      Web Audio API (AnalyserNode)
描画:          Canvas API (requestAnimationFrame)

ファイル構成
  music-visualizer/
    docs/
      仕様書.txt       この仕様書
    index.html          メインHTML
    style.css           スタイルシート
    app.js              アプリロジック


============================================================
LocalStorageキー
============================================================

music-vis-settings    設定 (mode, colorTheme, fftSize, sensitivity, bgAlpha)


============================================================
キーボードショートカット
============================================================

Space       音声取得の開始/停止
1〜5        ビジュアルモード切替
C           カラーテーマ切替
V           タブ映像背景のON/OFF
F           フルスクリーン切替
