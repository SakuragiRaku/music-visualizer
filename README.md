# 🎵 ミュージックビジュアライザー

スピーカーの音をリアルタイムに可視化するアプリ。

PCで再生中の音楽をタブ共有で取得し、幾何学模様としてCanvas上に描画。5種のビジュアルモードと4種のカラーテーマを搭載。

## 機能

- 🎧 タブ音声キャプチャ (getDisplayMedia API)
- 🎨 5種のビジュアルモード (バー・ウェーブ・サークル・パーティクル・万華鏡)
- 🌈 4種のカラーテーマ (レインボー・ネオン・モノクロ・オーシャン)
- ⌨️ キーボードショートカット (Space / 1-5 / C / F)
- 🎛 感度調整・背景トレイル・FFTサイズ設定

## 使い方

1. `index.html` をブラウザで開く
2. 「音声をキャプチャして開始」をクリック
3. 音楽を再生しているタブを選択して音声共有を有効に

## 技術

HTML / CSS / JavaScript / Web Audio API / Canvas API / getDisplayMedia API
